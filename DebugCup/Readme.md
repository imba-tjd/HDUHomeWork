## 比赛经历

5个小时，总共9道题，做出来三道= =。还做了另外两道，但分别是WA和栈溢出，记录在这里。

只能用C89是真的垃圾。还有同学觉得for loop外面声明循环变量是正常操作。

VSC的Debug需要DNF4.6.2然而机房电脑没装。只能用CB来调试了，Watch都要自己开，要不是我用过。

真的佛了。我只能说中国的程序设计竞赛只适合用C++。
